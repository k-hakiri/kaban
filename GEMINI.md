# 波切カバン店 Webサイト プロジェクトガイドライン

このファイルは、AIアシスタントが本プロジェクトのコードベースを理解し、一貫したルールで開発・保守をサポートするためのコンテキスト情報をまとめたものです。

## プロジェクト概要
山梨県甲府市にあるカバン修理専門店「波切カバン店」の公式ウェブサイト。
SPA（Single Page Application）風のスクロール遷移を採用しつつ、SEO対策のためにNode.jsを用いた静的サイト生成（SSG）を行っています。

## 技術スタック
- **フロントエンド**: HTML5, Vanilla CSS, Tailwind CSS（ユーティリティクラス）
- **ビルドツール**: Node.js (`build.js` による独自スクリプト)
- **インフラ/ホスティング**: Firebase Hosting

## 開発・デプロイフロー (必須ルール)
本プロジェクトでコードを変更する際は、以下のフローを必ず守ってください。

1. **ソースコードの編集**:
   - メインのコンテンツやUIの変更は、ルートディレクトリの `index.html` を編集します。（※生成先の `/faq/index.html` 等を直接編集しないこと）
   - 各ルートのSEOメタデータ（title, description, keywords, og:tags）や構造化データ（JSON-LD）の変更は `build.js` の `routeConfig` を編集します。
   - スタイルの変更は `/assets/css/main.css` またはHTML内のインラインスタイルを編集します。

2. **静的ファイルのビルド**:
   - 変更後は必ず以下のコマンドを実行し、各ディレクトリの静的HTMLと `sitemap.xml` を再生成してください。
   ```bash
   npm run build
   ```

3. **コミット**:
   - ソースコードの編集後やアセットの追加・変更、ビルドによる更新があった場合は、必ず変更内容をコミットしてください。
   - sitemap.xmlは指示がない場合は、コミットする必要はありません。

4. **デプロイ**:
   - ビルドおよびコミット完了後、以下のコマンドで本番環境に反映します。
   ```bash
   firebase deploy
   ```

## SEO・メタデータ管理の方針
- **ローカルSEOの重視**: 「山梨県」「甲府市」「甲斐市」「笛吹市」などの地域キーワードをメタデータや本文（主なご来店エリア等）に含めます。
- **ブランド名SEO**: 「ルイ・ヴィトン」「グッチ」「シャネル」などのハイブランド修理実績があることを、FAQやメタデータに明記しています。
- **構造化データ**: `LocalBusiness` スキーマを `build.js` 内で定義し、サービス提供エリア (`areaServed`) や店舗情報をGoogleに提供しています。
- **ファビコン**: 検索結果での表示互換性を高めるため、PNG, ICO, Apple Touch Icon 形式を利用しています。

## コンテンツのトーン＆マナー
- **誠実さと職人気質**: 80歳のベテラン職人による丁寧な手作業という強みを活かし、誇張せず誠実な表現（例：「修理内容によってはお受けできない場合もある」等）を用います。
- **透明性**: 見積もりは無料であることを明示しつつ、確認にお時間をいただく場合や手付金が発生するケースなど、実態に即した正確な情報を記載します。
- **顧客への配慮**: 「大切なカバンと長く歩めるように」というメッセージを軸に、ユーザーに安心感を与える丁寧な日本語を使用してください。
